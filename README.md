# ApiWay: Job
Dockerized TC(Test Case) running system.

## Hot it works
1. Docker run from [Dockerfile](https://github.com/apiway/apiway-job/blob/master/Dockerfile)
2. Authmatically clone TC project from github(or bitbucket)
3. Run TC project
